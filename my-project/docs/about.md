# about my hometown
我的家乡不是传统意义上的大。在上个世纪时是三线建设的受益者，而在之后便是大厦崩塌。现在就发展一个半死不

活的旅游业。扶贫所建的房子对与很多人来说也只是放假时的落脚点。长年都是在外读书，家乡对我来说就像是最熟

悉的陌生人。
